**WORKSCHEDFLOW App** es la extensión móvil oficial del ecosistema WORKSCHEDFLOW. Diseñada para ofrecer a los empleados una experiencia fluida y moderna al consultar su jornada laboral, horarios y registros directamente desde sus dispositivos móviles.

## 📱 Sobre el Proyecto

Esta aplicación móvil complementa la plataforma web para proporcionar movilidad y accesibilidad. El enfoque principal es la **claridad visual** y la **eficiencia**, transformando los datos de la API en una línea de tiempo intuitiva y profesional.

### ✨ Características Principales
* **Planificación Mensual**: Visualización clara de horarios de entrada, salida y duración total de la jornada.
* **Gestión de Días Libres**: Identificación visual inmediata de periodos de descanso con indicadores temáticos.
* **Sincronización API**: Conexión nativa con el backend de WORKSCHEDFLOW para datos en tiempo real.
* **UI/UX Profesional**: Diseño basado en la identidad visual de la web, optimizado para interacción táctil.

## 🛠️ Stack Tecnológico

* **Framework**: [Expo](https://expo.dev/) (React Native).
* **Lenguaje**: [TypeScript](https://www.typescriptlang.org/) para un desarrollo con tipado fuerte y seguro.
* **Estilos**: [NativeWind](https://www.nativewind.dev/) para una implementación de Tailwind CSS en entornos nativos.
* **Arquitectura**: Separación de capas (Infrastructure, Utils, UI) para facilitar el mantenimiento.
